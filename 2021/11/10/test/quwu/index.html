# 一、Retinex 理论

retinex 认为大气成像是由入射光函数和反射光函数，这两个乘积分量构成。即相机获得的图像 I (x,y), 由照明函数 A (x,y) 与反射函数 R (x,y) 的乘积。

 I(x,y)=A(x,y)∗R(x,y)①\begin {array}{c} \ I(x,y)=A(x,y)*R(x,y)\qquad ① \end {array} I(x,y)=A(x,y)∗R(x,y)①​

其中，反射分量反映了图像的实际内容，如细节、纹理、边缘等，在空间上快速变化，频谱在空间的高频区域。而照明分量在空间上缓慢变化，频谱在空间的低频区域。可以通过滤波将两者分开。

Retinex 认为：物体的颜色是由物体对长波、中波和短波光线的反射能力决定的，而不是由反射光强度的绝对值决定的。即物体的色彩不受光照的影响，具有一致性。

# 二、什么是大气物理散射模型？

大气物理散射模型在机器视觉领域中：大气散射模型应用于描述雾霾图像的退化过程。认为：采集的图像质量降低，由物体反射光在传播过程中，被传播路径内存在的微粒吸收散射光和光照在传播介质中散射的背景光造成。

认为：采集的图像质量降低，由场景内物体反射光在传播过程中，被传播路径内存在的微粒吸收散射光和光照在传播介质中散射的背景光造成。

简单来说，大气物体散射模型认为，所有光在传播过程中都会散射，接收到的图像都是原图损失过的。

 I(x,y)=J(x,y)∗t(x,y)+A(1−t(x,y))②\begin {array}{c} \ I(x,y)=J(x,y)*t(x,y)+A(1-t(x,y))\qquad ② \end{array} I(x,y)=J(x,y)∗t(x,y)+A(1−t(x,y))②​

上面公式里的 I (x,y) 代表相机接收到的图像，J (x,y) 代表物体实际的图像，t (x,y) 代表场景透射率，A 代表大气光。公式可以理解为：实际图像 J 经过散射后，衰变为 Jt。接收到的图像还由另一部分组成，没有散射的大气光：A*(1-t)

# 三、暗通道理论

在这里，暗通道的作用就是求场景透射率 t

可以了解一下他的推导过程。暗通道是对于一幅图像，在 r、g、b 三个维度取最小值，并对其邻域取最小值。在取这两次最小值之后，它的值基本可以视作 0，通过很多图像的实验证明，在取这两次最小值之后，这个值几乎为 0，或者很小，图像几乎全黑。
